# booking-platform

The live, continuously-evolved monorepo for the Production Project Architecture module.

This folder is empty until Lesson 5 ("Building Your First Monorepo"), then grows lesson by lesson:

- Lesson 5: `apps/web`, `apps/api`, `packages/shared-types`, `packages/utils` + pnpm workspaces + Turborepo
- Lesson 6: `packages/ui`, `packages/auth`
- Lesson 7: `apps/api` refactored into `modules/`
- Lesson 8: `apps/worker`
- Lesson 9: RBAC enforcement in modules
- Lesson 10–11: split into `booking-service` / `notification-service` / `payment-service` + RabbitMQ
- Lesson 12: Dockerfiles + CI/CD
- Lesson 13–14: final structure + capstone write-up
